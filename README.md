# Hospital_server
Web application for hospital workers (i.e. nurses and doctors).

User can:
1) Add a patient;
2) Update;
3) Delete; 
4) Get;
5) Change style of their app(different color of background).

Tools:
1) HTML (Bootstrap)
2) CSS 
3) AJAX
4) JavaScript
5) Jquery 
6) Node.js
7) Express 
8) MongoDB
